## Todo List App Docs


### Installation (Windows)

1. **Clone the Repository:**

   ```bash
   git clone https://github.com/PiyushGhavghave/TODO-app.git
   ```

2. **Navigate to the Project Directory:**

   ```bash
   cd TODO-app
   ```

3. **Open `index.html` in a Browser:**

   ```bash
   start index.html
   ```
